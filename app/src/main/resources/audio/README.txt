# 음악 파일 위치

이 폴더에 다음 WAV 파일들을 넣어주세요:

1. menu_theme.wav - 메인 메뉴 배경음악
2. game_theme.wav - 게임 플레이 배경음악 (테트리스 클래식 음악)
3. battle_theme.wav - 대전 모드 배경음악

## 빠른 시작

음악 파일을 구하는 가장 쉬운 방법:

1. 유튜브에서 "Tetris Theme A Korobeiniki" 검색
2. 온라인 변환기로 WAV 변환: https://online-audio-converter.com/
3. 이 폴더에 파일 복사
4. 게임 실행!

음악 파일이 없어도 게임은 정상적으로 작동합니다.
게임 중 M 키를 눌러 음소거할 수 있습니다.
